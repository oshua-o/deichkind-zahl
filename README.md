## Deichkind-Zahl herausfinden leicht gemacht:
Statt langem Rumrätseln und auf Wikipedia-Seiten rumklicken und dabei 90 % der Artists verpassen gibt es jetzt das ultimative Tool!
Mit MusicBrainz (https://musicbrainz.org/) kann man direkt herausfinden, wer mit wem zusammengearbeitet hat.
Der Code, der das dann weiter verarbeitet, ist zwar stundenlang von meinen Ideen gefüttert worden, aber von Sonnet 4.6 (claude.ai) dann realisiert worden, da mir die handwerklichen Fähigkeiten ausgegangen sind. Immerhin funktionierts jetzt.

## Was muss ich tun?
- deichkind-zahl.py herunterladen
- Python installiert haben
- pip install musicbrainzngs
- Los geht's!

## Warum gibt es das?
Weil es wichtig ist, zu wissen, welche Deichkind-Zahl welche*r Musiker*in hat.
